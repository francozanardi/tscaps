@tscaps/engine examples

Four minimal consumers of the engine's render pipeline:

  • From text — captions come from a textarea, no transcription. Fastest path to a rendered video; great first taste of the engine.
  • Transcribe — same defaults as above, but captions come from the in-browser Whisper transcriber over the bundled clip's audio.
  • CSS + alignment — same as transcribe with a custom stylesheet and top-aligned captions.
  • CLI — driven from Node via Playwright; run pnpm cli in this package.
